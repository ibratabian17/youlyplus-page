Features that fix the experience

We didn't just skin the player. We rebuilt the rendering engine to fix broken native implementations and added features the platforms forgot.

The Apple Music Web Fix

The native Apple Music web player is heavy, buggy, and drifts out of sync. YouLy+ replaces the resource-heavy native UI with a lightweight, custom renderer that ensures frame-perfect sync without hogging your CPU.

True Word-by-Word

Tidal Web only offers line sync. YouTube Music is inconsistent. YouLy+ upgrades both to syllable-level, word-by-word synchronization using Apple TTML data for a true karaoke experience.

Performance First

Built to run where native players fail. Optimized for 60 FPS animations even on older hardware (dual-core CPUs/older GPUs). Choose between Lightweight or Compatibility modes to fit your rig.

AI & Romanization

Instantly Romanize K-Pop or J-Rock. Translate lyrics on the fly using Google Translate, or connect your own Gemini AI / OpenRouter API key for context-aware LLM translations.

SponsorBlock Ready

YouLy+ intelligently detects non-music segments on YouTube Music (intros, outros, sponsor spots) so your lyrics never drift out of sync when you skip the filler.
