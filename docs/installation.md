## Method 2: Manual Installation
**Best for: Chrome, Brave, Vivaldi**

Since YouLy+ is not yet on the Chrome Web Store, you must install it manually ("sideloading").

### 1. Download & Extract
1.  Visit the **[YouLy+ Releases Page](https://github.com/ibratabian17/YouLyPlus/releases)**.
2.  Find the **Latest** version.
3.  Under **Assets**, download the zip file labeled for Chrome (e.g., `youlyplus-vX.X.X-chrome-edge.zip`).
4.  **Extract (Unzip)** the file to a **safe, permanent location** (like your Documents folder).
    > ‚ö†Ô∏è **Important:** Do not delete this folder. Chrome loads the extension directly from these files.

### 2. Load into Browser
1.  Open your browser and enter `chrome://extensions` in the address bar.
2.  Toggle **Developer mode** on (usually in the top-right corner).
3.  Click the **Load unpacked** button that appears.
4.  Select the **folder** you extracted in the previous step.

YouLy+ is now active! üéâ

## üîÑ How to Update (Manual Install)

Manually installed extensions do not update automatically. To update:

1.  Download the new `.zip` from the [Releases Page](https://github.com/ibratabian17/YouLyPlus/releases).
2.  Extract the files and **overwrite/replace** the files in your existing installation folder.
3.  Go to `chrome://extensions` and click the **Reload** icon ‚ü≥ on the YouLy+ card.
4.  Refresh YouTube Music to see the changes.

## üõ†Ô∏è Troubleshooting

**"Manifest file is missing or unreadable"**
*   This usually happens if you select the `.zip` file instead of the *folder*. Ensure you unzip the file first, then select the extracted folder via "Load unpacked".

**Extension not appearing or working**
*   Ensure the toggle switch on the `chrome://extensions` card is **ON**.
*   Perform a **Hard Refresh** on YouTube Music (`Ctrl + Shift + R` or `Cmd + Shift + R`).
*   Restart your browser.
